## 📄 What is geodesic-wallpaper?

geodesic-wallpaper creates real-time animated wallpapers for Windows desktops. It shows moving patterns based on geodesic flows. Geodesic flows come from math about curves and surfaces, used here to create soothing visuals. This app runs smoothly using your computer’s graphics hardware.

## 🖥️ About the technology

This wallpaper uses the Rust programming language for fast performance and safety. It uses `wgpu`, a graphic library designed for modern PCs to work efficiently with the GPU hardware. This lets the wallpaper render smooth curves and dynamic shapes without slowing your system.

The visuals come from mathematical concepts called geodesic flows. These are paths on curved surfaces similar to great circles on a globe. You see flowing lines and patterns that move naturally over these shapes.
